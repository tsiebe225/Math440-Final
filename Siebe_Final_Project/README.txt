The program should be able to run after using the makefile and the regular execution with the slurm file or using regular commands at a sayers lab machine. The program will output ranges of values for various options to help evaluate american future options. The scaling exercises were completed by altering the variable timeSpanNum to whatever was desired. The analysis excel doc contains all the of the runtimes that were found. Some of the .out files were overwritten after the excel document and analysis was completed but the values were still within reason and close to one another. The parameters are easy to alter in order to model a different asset/stock.
